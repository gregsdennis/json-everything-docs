# PathScope Enum

Namespace: Json.Path

Indicates the scope of a path.

## Values

| Name | Summary |
|---|---|
| **Global** | The scope is the entire JSON document.  These paths start with `$`. |
| **Local** | The scope is the local JSON value.  These paths start with `@`. |
